Tool Tracker v1.1

A command-line Python application for managing tools and skills.

Features

- Add new tools
- View all tools
- Search for a tool
- Update tool information
- Remove tools
- Save tools to a file
- Load tools from a file
- Menu-driven interface

Technologies Used

- Python
- File Handling
- Lists
- Dictionaries
- Loops and Conditions

How to Run

1. Clone the repository.
2. Navigate to the project folder.
3. Run:

python tool_tracker.py
